MESSAGE API

Welcome to the MESSAGE API, with this API you can manage the users and their messages on your website, in this guide you will find explanations on how to use the API features and implement them. This API is built in node.js

Domain: https://api-message.onrender.com

User

You need to log in to the system to be able to send messages and view your messages, sending messages is only possible to users registered in the system
Method URL Body Description
GET
 /user Get all users - by admin token.
GET
 /user/myInfo Get your user information - by Token
POST
 /user 
{ 
   name: {
      firstName: Rivka,
      lastName: Shabtay
   },
   "email": "rivki3232@gmail.com",
   "password": "rivki3232"
}
Add a new user
POST

Update user, You can only update your user. And admin can update all users
DELETE
 /user/id Delete a user - by admin token.

Message

You need to log in to the system to be able to send messages and view your messages, sending messages is only possible to users registered in the system
Method URL Body Description
GET
 /message/sent Get all messages sent - by Token.
GET
 /message/inbox Get all received messages - by Token
GET
 /message/sent/?perPage=5&page=2

/message/inbox/?page=1&sort=title&reverse=yes 		You can send URL queries to display the data.

perPage: amount of messages per page.

page: page Number.

sort: sort by.

reverse: sort in reverse order.
GET
 /message/sent/single/{id} Get a single message sent by id and token
GET
 /message/inbox/{idFrom} Get all messages from a specific user by user id and token
GET
 /message/sent/search/?s=mes

/message/inbox/search/?s=mes 		Search for a message by searching the title or body of the message after the "s=", it will show 10 results per page.

 /message 
{

DELETE
